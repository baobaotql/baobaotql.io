ISSUE · 01
JSON 解析多层容错
AI API 返回格式不稳定，5 种策略保障：直接解析 → 去 markdown → 贪婪正则 → 平衡括号匹配 → 错误修复。极端情况永不崩溃。
ISSUE · 02
低风险安全语言过滤
防止 AI 输出"内容安全"等绝对描述误导老人。8 个禁止词 + 替代文案。低风险必须写"暂未发现明显风险特征"。
ISSUE · 03
组合升级规则设计
中风险关键词 ≥ 3 条同时命中自动升级高风险。宁可误判为高也不能漏判，这是针对老年群体的安全底线。
